# m-lgw-lab

这是一个用于归档和发布多个独立 HTML 页面的仓库。  
仓库内容以静态页面为主，不针对任何特定域名或网站，仅作为页面资源的集中存放与版本管理使用。

## 项目简介

本仓库的主要目的是整理、保存和展示一些独立开发的 HTML 页面。  
这些页面可能包括工具页面、演示页面、原型页面或其他轻量级前端内容。  
仓库本身不提供任何指向外部资源的链接，也不包含营销或推广性质的信息。

## 目录说明

```
.
├── index.html          # 仓库入口页面（可选）
├── pages/              # 存放各独立 HTML 页面
│   ├── page-01.html
│   ├── page-02.html
│   └── ...
└── assets/             # 页面使用的静态资源（CSS、JS、图片等）
    ├── css/
    ├── js/
    └── images/
```

- **pages/**：存放所有独立 HTML 页面文件，每个页面为一个独立单元。
- **assets/**：存放页面共用的或各自引用的样式、脚本、图片等资源文件。

## 页面归档说明

- 每个 HTML 页面均为独立完整的文件，可直接在浏览器中打开使用。
- 页面之间无依赖关系，可单独查看或部署。
- 页面内容可能涉及工具、演示、实验性功能等，不承诺长期更新或维护。
